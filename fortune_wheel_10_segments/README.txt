# Roue de fortune — 10 zones

Fichiers prêts à héberger pour une roue cliquable (tourner / arrêter) utilisant **votre image**.

## Utilisation
1. Hébergez `index.html` et `wheel.png` (GitHub Pages, Netlify, Vercel, ou votre site).
2. Ouvrez `index.html` dans un navigateur pour tester en local.
3. Dans Genially : **Insérer → Autres → Insérer (contenu web)** puis collez l'URL publique.

## Personnalisation
- Modifiez les libellés dans `LABELS` (10 éléments).
- Ajustez la taille en changeant `--size` dans le CSS.
- Vitesse / freinage : `MAX_SPEED`, `FRICTION`, `RANDOM_BONUS` dans le script.

Bonne animation! 🎉
